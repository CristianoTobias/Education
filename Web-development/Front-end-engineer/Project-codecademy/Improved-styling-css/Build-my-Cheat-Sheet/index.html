Tag's HTML5

Tg Name Description
<!DOCTYPE html> Doctype tells the browser the version of HTML being used in the document.
<html> HTML This tag is the basic element of the HTML structure. Therefore, it will contain all the elements of your document.
<head> Head This tag delimits the document header. It has no visible value, but it is capable of transmitting to the browser several very useful and essential information for a good presentation of your HTML document.
<title> Title This tag defines the title of your page, the name that will appear on your tab, window or tab. For this reason, the >title> tag is very important for SEO;
<meta> Meta This tag allows you to insert metadata into your document, in the case above, the information charset=”UTF-8″, which guarantees code compatibility with Latin American characters;
<body> Body Finally, the tag that represents the body of the document. In summary, this tag is where all the visible elements of your site must be inserted.
<!----> Comentário All elements included within it will be just comments, that is, they will not be visible in the browser
<header> Header Content inside it is part of a header and can be used within other sessions.
<main> Main They represent the main content of its body, that is, the content directly related to the central topic of the page or with the central functionality of the application;
<footer> Footer These tags define a footer for the page, usually used at the end of the page;
<section> Section These tags define a session for your page;
<article> Article These tags define an article on your page. In this sense, they are used to separate the content of your page. Widely used mainly by blogs or content pages;
<aside> Aside These tags represent a section of a page whose content is tangentially related to the surrounding content, which could be considered separate from the content;
<nav> Nav This tag defines navigational content. Therefore, it is widely used in conjunction with lists and in creating menus;
<div> Div Defines a page division. In this way, it works as a container for stream content. Since it has no semantic value, it is often used to better organize content.
<h1><h6> Titles Header elements are implemented at six levels, and describe the topic of the section it is in.
<p> Paragraph Main text tag, makes up a paragraph.
<span> span Used only for small information such as form captions, image captions, etc. It can also be used to form a container
<pre> Pre Tag used to represent preformatted text. Widely used to enter codes;
<b> Bold Makes the content bold;
<i> Italic Makes the content italic;
<br> Line break This tag does not need closing, it performs the line break function.
<hr> Horizontal Rule This tag does not need a closing, it forms a horizontal line.
